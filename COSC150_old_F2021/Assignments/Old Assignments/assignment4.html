The goal of this assignment is for you to explore and discover the power and limits of numerical modeling.

  1. Review your notes from class on features of Excel. In particular you should know how to:

    • Name a cell as a variable
    • Name a range of cells as a variable
    • Introduce a control such as a scroll bar
    • Then use a scroll bar to assign a non-integer value between two limits
    • Use an IF statement to test some condition, and set a value if true, and a different value if false.

  2. Demonstrating as appropriate your skills above, construct an Excel file in which you:

    • Find the largest positive integer that Excel can represent as an integer
    • Find the most negative integer that Excel can represent as an integer
    • Find the largest floating point number that Excel can represent
    • Find the most negative floating point number that Excel can represent
    • Find the smallest positive floating point number that Excel can represent
    • Find the largest number that, when added to 1, still returns 1
    • Find the largest number that, when added to Avogadro's Number, still returns Avogadro's Number
    • Find the largest number that when you take the cosine of that number still returns 1
    • Find the largest integer for which FACT(N) accurately returns N! (N factorial)
